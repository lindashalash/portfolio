ParcelEye: AI-Powered Parcel Tracking System

📦

ParcelEye is a cross-platform parcel tracking system developed to mitigate theft at unattended collection points. Designed with modularity, real-time AI, and seamless mobile integration, the system uses computer vision to track parcels and send immediate push notifications when packages become untraceable. The application empowers users to monitor their parcels with high accuracy and reliability, acting as a deterrent against crimes of opportunity. Features include Real-time AI tracking using Meta's SAM 2.1 model to detect parcel presence Push notifications via Firebase when parcels are lost or re-acquired in frame Secure user authentication with email verification and session token management Live camera stream via HLS, protected with dynamic access control Modular system architecture comprising Android App, Spring Boot API, and Flask-based AI server Deployment ready using Docker containers and hosted on AWS

Full-Stack Development (Python) Android (Java/Kotlin) Computer Vision (OpenCV) Spring Boot (Java) Firebase Cloud Messaging Docker & Containerization RESTful API Design AI Integration (Meta SAM2.1) MongoDB (NoSQL) Cloud Deployment (AWS)

InQuery

📚

InQuery is a web-based platform developed as part of the Undergraduate Research Opportunities Programme (UROP) at SUTD. The project explores how live interactive querying—powered by Large Language Models (LLMs) like GPT-4, can support and enhance learning in classroom environments. By integrating real-time transcription of lectures with GPT-4, InQuery allows students to engage directly with lecture content by asking clarifying questions, receiving context-aware explanations, and improving note-taking in real time. The system is built using Flask for both the front-end and back-end, with a responsive UI built using HTML, CSS, JavaScript, and Twitter Bootstrap. The result is an intelligent, interactive educational tool designed to foster deeper understanding and more active classroom participation.

Flask (Python) HTML CSS JavaScript Twitter Bootstrap GPT-4 via OpenAI API RESTful API
