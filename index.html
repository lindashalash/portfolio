Project Overview

The Ascenda Hotel Booking Website is a comprehensive full-stack web application developed in collaboration with Ascenda, a leading technology company in the travel industry. This project showcases modern web development practices, user-centered design, and seamless API integration.

Key Features

User Authentication: Secure login and registration system with JWT tokens and session management.

Hotel Search & Booking: Advanced search functionality with filters for location, price, amenities, and dates.

Payment Processing: Integrated payment gateway for secure transactions and booking confirmations.

Admin Dashboard: Comprehensive management interface for hotel administrators to manage bookings, inventory, and analytics.

Responsive Design: Mobile-first design approach ensuring optimal experience across all devices.

Technical Implementation

Built using React for the frontend with modern hooks and context API for state management. The backend leverages Node.js with Express framework, connecting to MongoDB for data persistence. Docker containerization ensures consistent deployment across environments, while Tailwind CSS provides utility-first styling approach.

Team Collaboration

This project emphasized collaborative development using Git feature branching, code reviews, and agile methodologies. Regular standups and sprint planning sessions ensured efficient progress and quality deliverables. UI/UX design was prototyped using Figma with iterative feedback from stakeholders.

Challenges & Solutions

One major challenge was integrating Ascenda's complex hotel service API while maintaining fast response times. We implemented caching strategies and optimized API calls to ensure smooth user experience. Another challenge was handling concurrent booking requests, which we solved using database transactions and proper error handling.

ParcelEye: AI-Powered Parcel Tracking System

📱
🤖
🔔
☁️

Project Overview

ParcelEye addresses the growing problem of parcel theft at unattended collection points through innovative AI-powered surveillance. The system combines computer vision, real-time processing, and mobile notifications to create a comprehensive security solution that acts as both a deterrent and early warning system.

AI & Computer Vision

Meta SAM 2.1 Integration: Utilizes the latest Segment Anything Model for precise object detection and tracking.

Real-time Processing: OpenCV-based pipeline processes video streams with minimal latency for immediate threat detection.

Adaptive Learning: The system learns parcel characteristics and improves accuracy over time through continuous monitoring.

Environmental Adaptation: Robust performance across different lighting conditions and weather scenarios.

System Architecture

The modular architecture consists of three main components: an Android mobile application for user interaction, a Spring Boot API server for business logic and authentication, and a Flask-based AI server for computer vision processing. This separation ensures scalability and maintainability while allowing for independent deployment and updates.

Mobile Application Features

Real-time Notifications: Instant push notifications via Firebase Cloud Messaging when parcels are detected as missing.

Live Camera Feed: Secure HLS streaming with dynamic access control for remote monitoring.

User Authentication: Robust security with email verification and session token management.

Intuitive Interface: Clean, user-friendly design focused on essential functionality and quick access to alerts.

Cloud Infrastructure

Deployed on AWS with Docker containerization for consistent performance and easy scaling. The system handles multiple concurrent video streams while maintaining low latency for real-time alerts. MongoDB provides flexible data storage for user information, device configurations, and tracking history.

Impact & Results

ParcelEye demonstrates the practical application of AI in solving real-world security challenges. The system's modular design makes it easily adaptable to different environments and scalable for commercial deployment. Testing showed high accuracy in parcel detection with minimal false positives, making it a viable solution for reducing package theft.

InQuery: Interactive Learning Platform

🎓
💬
📝
🧠

Project Overview

InQuery is a groundbreaking research project developed under the Undergraduate Research Opportunities Programme (UROP) at SUTD. The platform explores how Large Language Models can revolutionize classroom learning through real-time, context-aware interactions that enhance student engagement and comprehension.

Research Objectives

Educational Enhancement: Investigate how AI-powered tools can supplement traditional teaching methods.

Real-time Interaction: Enable students to ask clarifying questions during lectures without disrupting the flow.

Context Awareness: Provide answers that are specifically relevant to the current lecture content and context.

Learning Analytics: Collect data on student engagement patterns and common areas of confusion.

Technical Implementation

The platform is built using Flask as both the web framework and API backend, providing a lightweight yet powerful foundation. The frontend combines HTML5, CSS3, and JavaScript with Twitter Bootstrap for responsive design and intuitive user interactions. GPT-4 integration via OpenAI's API enables sophisticated natural language processing and contextual responses.

Key Features

Live Transcription: Real-time conversion of lecture audio to text for context-aware question answering.

Intelligent Q&A: Students can ask questions that are answered based on current lecture content and broader educational context.

Note-taking Enhancement: Automatic generation of key points and summaries to support student note-taking.

Interactive Interface: Clean, distraction-free design that integrates seamlessly into the classroom environment.

Educational Impact

InQuery addresses common classroom challenges such as students hesitating to ask questions publicly, difficulty keeping up with fast-paced lectures, and varying levels of background knowledge among students. By providing a private, AI-powered assistant, students can get immediate clarification without interrupting the class flow.

Research Outcomes

The project demonstrates the potential for AI to enhance educational experiences without replacing human instruction. Initial testing showed improved student engagement and better comprehension of complex topics. The research contributes to the growing field of AI in education and provides insights for future educational technology development.

Future Development

Future iterations could include multilingual support, integration with learning management systems, and advanced analytics for educators to identify common knowledge gaps. The platform's modular design allows for easy expansion and adaptation to different educational contexts.
